# Abbey OS v2

A Bloomberg-style dashboard for managing Portfolio, UK Debt, Hospitality, and F&B operations.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Portfolio Dashboard**: Comprehensive portfolio management and analytics
- **UK Debt**: UK debt tracking and management
- **Hospitality**: Hospitality operations dashboard
- **F&B**: Food & Beverage operations management

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18
